# CommentTag — 유저 태그 기능

- **기능명**: 댓글 유저 태깅
- **담당자**: 최민제

---

## 기능 설명

댓글 입력 중 `@` 문자를 입력하면 닉네임 검색 드롭다운이 나타나고, 유저를 선택해 태그할 수 있습니다.

---

## 사용자 흐름

1. 댓글 입력창에서 `@` 입력
2. 유저 검색 드롭다운 노출 (닉네임 자동완성)
3. 유저 선택 시 `@닉네임` 이 입력창에 삽입됨
4. 댓글 제출 시 `content` 문자열에 `@닉네임` 포함하여 전송

---

## API

### 유저 검색 (태그 자동완성)

- **Method:** `GET`
- **Endpoint:** `/api/v1/users/search?nickname={query}`
- **인증:** 필요 (`Bearer Token`)
- **호출 시점:** `@` 입력 후 닉네임 쿼리가 변경될 때마다

**성공 응답 (200)**

```json
{
  "results": [
    {
      "id": 2,
      "nickname": "testuser2",
      "profile_img_url": "https://example.com"
    },
    {
      "id": 3,
      "nickname": "testuser3",
      "profile_img_url": "https://example.com"
    }
  ]
}
```

**실패 응답**

```json
401: { "error_detail": "자격 인증 데이터가 제공되지 않았습니다" }
```
